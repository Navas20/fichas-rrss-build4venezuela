# Fichas RRSS — Build4Venezuela

Generador de fichas de búsqueda 1:1 para Instagram y WhatsApp. Crea fichas estandarizadas para ayudar en la difusión de personas desaparecidas durante la emergencia en Venezuela.

## Uso

1. Abre `index.html` en tu navegador o desde un servidor local
2. Completa los datos: foto, nombre, edad, ubicación, contacto
3. Haz clic en "Generar ficha"
4. Descarga el PNG o copia la imagen al portapapeles
5. Comparte en Instagram, WhatsApp o redes sociales

## Características

- Generación de fichas 1:1 (1080x1080px) listas para redes sociales
- Foto con recorte circular y fallback con iniciales
- Tema oscuro/claro automático según preferencia del sistema
- Validación de campos obligatorios y formato de imagen
- Persistencia de datos en localStorage (no se pierden al recargar)
- Descarga PNG y copia al portapapeles (con fallback de texto)
- Drag & drop para subir foto
- Accesibilidad: navegación por teclado, ARIA labels, contraste WCAG
- Responsive: funciona en desktop y mobile

## Tecnologías

HTML5, CSS3, JavaScript vanilla, Canvas API
